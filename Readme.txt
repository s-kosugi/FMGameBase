/*  FMGameBaseについて  */

◎はじめに
このライブラリはDXライブラリのラッパーライブラリです。
Windows用の2Dゲーム制作用のライブラリとして作成しております。
C++でのゲームプログラム教育用として作成しました。


---------------------------------------------------------------------------
◎動作について
Windows10にて動作確認をしております。
VisualStudio2019+DXライブラリ+Effekseerが動く環境であれば動かす事ができます。
DXライブラリの使い方やEffekseerについては各サイト様を参照ください。

①DXライブラリの配置
DXライブラリ様のサイト
https://dxlib.xsrv.jp/dxdload.html
からWindows版VisualStudio2019をダウンロードし、
"プロジェクトに追加すべきファイル_VC用"フォルダ内部にある全てのファイルを
FMGameBaseと同階層のフォルダ"DXLib_VC"というフォルダに全てコピーしてください。

②Effekseerライブラリの配置
Effekseer様のサイト
http://effekseer.github.io/jp/download.html
から"Effekseer For DXライブラリ"をダウンロードし、
"プロジェクトに追加すべきファイル_VC用"フォルダ内部にある全てのファイルを
"①"で作ったフォルダ内部に上書き保存してください。

③FMGameBase.slnを実行し、実行する事でゲーム画面が表示されます。
---------------------------------------------------------------------------
◎プログラム全体について
オブジェクトの管理は木構造を採用しており、動的に生成されたオブジェクトを子クラスリストに登録しています。
木構造で管理したいオブジェクトはIBaseObjectを継承し、オブジェクト生成時にはCreateObject命令を使って動的にオブジェクトを生成して子に追加するようにしております。
また、DeleteObject命令で子を含む自身のオブジェクトを削除できるようになっている為、メモリ解放忘れを防ぐようにしております。
---------------------------------------------------------------------------
◎描画関係について
また、描画関係については各オブジェクトの処理とは独立させて描画優先を数値で設定できるようにしてあります。
詳しい説明についてはdocsフォルダのDrawCtrl解説と描画オブジェクトについてをご参照ください。
---------------------------------------------------------------------------
◎アニメーション再生について
json形式で書かれた.assというファイルを読み込む事で、アニメーションスプライトを実装できるようにしてあります。
cAnimSpObjectを継承してクラスを作成し、生成時に.assファイルを読み込むことでアニメーションスプライトクラスを生成できます。
dataフォルダ内部にplayer_01.assというファイルがサンプルとして置いてあります。
---------------------------------------------------------------------------
◎エフェクト再生について
Effekseer再生ライブラリを取り込んでいますので、外部ツール(Effekseer)で作られたエフェクトをゲーム上で綺麗に簡単に再生することができます。基底クラス"cEffectObject"を継承する事で、ファイル名を指定して生成するだけですぐにエフェクトが再生されます。

---------------------------------------------------------------------------
◎サーフェイス処理描画について
cSurfaceオブジェクトを利用することで、複数の描画オブジェクトを１つのバッファにまとめ、１オブジェクトとして描画することもできます。画面全体に効果をかけたい時やマルチプレイ画面を実装するとき等に利用してください。


---------------------------------------------------------------------------
◎トランジションについて
ルール画像を用いてのシーン遷移にも対応してあり、cSceneManagerにあるChangeSceneUniTransを使えば様々なシーン切り替え効果を実装することもできます。


他にもゲーム開発をするのにあたって様々な便利な機能を実装しておりますので是非一度ご確認頂ければと思います。
